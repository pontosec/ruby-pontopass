Overview

Descrição

A classe PontopassUser é utilizada para gerenciamento dos dados de usuários no Pontopass. O Pontopass é um sistema para fácil implementação de autenticação em duas etapas, ou autenticação forte. Para utilizar o sistema, é necessário primeiro se cadastrar em pontopass.com. Para maiores detalhes, acesse pontopass.com e pontosec.com.
